# Simple animated GIF
<strong>Peek:</strong> https://github.com/phw/peek

You can install the latest versions of Peek from the Ubuntu PPA.

sudo add-apt-repository ppa:peek-developers/stable<br>
sudo apt update<br>
sudo apt install peek
